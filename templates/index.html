如何获取 cookies.txt
1

安装浏览器扩展

在 Chrome / Edge 中安装 Get cookies.txt LOCALLY 扩展。

Firefox 用户可搜索 "cookies.txt" 扩展安装。

2

登录目标网站

在浏览器中打开 bilibili.com 或 youtube.com，确保已登录你的账号。

3

导出 Cookie

在目标网站页面上，点击扩展图标，选择 「Export」 或 「当前站点」，将 Cookie 保存为 cookies.txt 文件。

建议分别在 Bilibili 和 YouTube 页面各导出一次，合并到同一个文件中。

4

上传到本工具

将导出的 cookies.txt 拖拽到上方的上传区域，或点击选择文件上传。

1

使用 yt-dlp 直接导出

如果你已安装 yt-dlp，可以在命令行中运行以下命令直接从浏览器导出 Cookie：

yt-dlp --cookies-from-browser chrome --cookies cookies.txt --skip-download "https://www.bilibili.com"

将 chrome 替换为你使用的浏览器：edge、firefox、opera、brave 等。

2

使用 Python 脚本导出

在项目虚拟环境中运行以下命令，自动从 Chrome 导出 Cookie：

python -c "import yt_dlp; yt_dlp.YoutubeDL({'cookiesfrombrowser':('chrome',),'cookiefile':'cookies/cookies.txt','skip_download':True}).extract_info('https://www.bilibili.com',download=False)"

此方法需要关闭浏览器后执行，否则可能无法读取加密的 Cookie 数据库。

3

上传导出的文件

命令执行后会在当前目录生成 cookies.txt，将其上传到本工具即可。

注意事项
  • Cookie 文件包含你的登录凭证，请妥善保管，不要分享给他人。
  • Cookie 有有效期，过期后需要重新导出上传。
  • Bilibili 大会员专属分辨率需要大会员账号的 Cookie。
  • 标记为 [V only] 的格式为纯视频流，下载时会自动合并最佳音频。
  • 下载的文件保存在服务器的 downloads 目录中。
  • 本工具仅供个人学习使用，请尊重视频创作者的版权。
